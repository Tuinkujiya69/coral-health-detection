# 🪸 Coral Health Detection System

A Flask-based web application that uses a Deep Learning model to detect whether a coral is **Healthy** or **Bleached** from an uploaded image.

---

## 📌 Project Overview
Coral reefs are very important for marine ecosystems, but they are affected by coral bleaching due to climate change and pollution.  
This project uses **Machine Learning and Computer Vision** to automatically detect coral health from images.

---

## 🚀 Features
- Upload coral image
- Detect whether coral is **Healthy** or **Bleached**
- Shows prediction with confidence score
- Simple and user-friendly web interface
- Built using Flask
